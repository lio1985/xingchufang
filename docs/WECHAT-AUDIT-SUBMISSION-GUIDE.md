## 二、审核说明文案（复制到"审核说明"框中）

### 标准审核说明

```
本小程序为企业内部内容创作工具，仅供授权员工使用。

【应用定位】
为企业内容创作团队打造的智能化内容创作平台，包含智能选题生成、内容生成与拆分、语料优化、爆款复刻四大核心系统。

【核心功能】
1. 智能选题生成：基于工作生活场景生成选题，辅助内容策划
2. 内容生成与拆分：智能生成脚本内容，多版本拆分对比
3. 语料优化：提供企业语料库、个人IP语料库、产品知识库，文本智能优化
4. 爆款复刻：分析热门内容，提供创作参考

【使用范围】
- 仅限公司内部授权员工使用
- 通过微信授权登录 + 白名单机制控制访问权限
- 管理员统一管理用户账号和权限
- 员工离职可立即取消权限

【权限控制机制】
1. 微信授权登录（确保身份真实性）
2. 用户白名单机制（只有白名单用户可登录）
3. 基于角色的权限控制（普通用户/管理员）
4. 数据隔离（用户只能查看自己的数据）
5. 操作日志记录（所有操作可追溯）

【数据安全】
- 所有业务数据绑定用户ID
- 敏感操作需要管理员权限
- 完整的审计日志功能
- 定期备份与监控

【管理后台】
提供完善的管理后台，管理员可以：
- 查看和管理所有用户
- 启用/禁用用户账号
- 配置语料库共享权限
- 查看统计数据和操作日志
- 导出用户数据

【合规说明】
本工具符合《微信小程序平台运营规范》，已配置用户协议和隐私政策，所有用户数据仅用于企业内部内容创作，不会对外共享或商业化使用。

【测试账号】
如需测试，请联系管理员获取测试账号。
```

## 四、审核注意事项

### 4.1 常见审核失败原因

| 原因 | 解决方案 |
|------|----------|
| 功能不完整 | 确保所有功能都能正常使用 |
| 界面不规范 | 检查是否有错位、显示异常 |
| 缺少用户协议 | 上传用户协议和隐私政策 |
| 类目不正确 | 确保选择了"工具 > 办公"类目 |
| 截图不清晰 | 使用真实、清晰的截图 |

### 4.2 特别说明

**关于"内部使用"的说明：**
- ✅ 微信允许企业内部工具类小程序上线
- ✅ 关键是要有完善的权限控制机制
- ✅ 在审核说明中明确说明使用范围
- ✅ 提供管理后台截图证明权限管理能力

**关于"审核账号"的建议：**
- 可以提供一个测试账号供审核人员使用
- 测试账号可以是普通用户权限
- 确保测试账号有完整的使用权限

## 五、审核通过后的操作

### 5.1 设置服务器域名

登录微信公众平台，设置以下服务器域名：
- **request合法域名：** https://your-domain.com
- **uploadFile合法域名：** https://your-domain.com
- **downloadFile合法域名：** https://your-domain.com

### 5.2 配置隐私设置

- 设置用户隐私保护指引
- 配置地理位置权限（如需要）
- 配置手机号权限（如需要）

### 5.3 发布正式版

1. 上传代码（上传生产环境构建版本）
2. 提交审核（填写审核说明）
3. 审核通过后发布

## 六、审核不通过的应对

### 6.1 分析不通过原因

根据微信审核团队反馈的原因进行针对性修改：
- 如果是功能问题 → 修复功能后重新提交
- 如果是文案问题 → 修改文案后重新提交
- 如果是权限问题 → 完善权限控制后重新提交

### 6.2 重新提交流程

1. 根据反馈修改代码或文案
2. 重新构建并上传代码
3. 补充审核说明（解释已修改的内容）
4. 重新提交审核
