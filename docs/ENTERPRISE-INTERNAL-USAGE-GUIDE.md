# 星厨房内容创作助手 - 企业内部使用说明

## 一、应用定位

**应用名称：** 星厨房内容创作助手
**应用类型：** 企业内部内容创作工具
**使用范围：** 仅限公司内部授权员工使用
**服务类目：** 工具 > 办公

## 二、核心功能

本工具是为公司内容创作团队打造的智能化内容创作平台，包含以下四大核心系统：

### 1. 智能选题生成系统
- 基于工作生活场景生成选题
- 支持多个话题方向推荐
- 辅助内容策划

### 2. 内容生成与拆分系统
- 智能生成脚本内容
- 多版本拆分对比
- 提升内容创作效率

### 3. 语料优化系统
- 语料库管理（企业语料库、个人IP语料库、产品知识库）
- 文本智能优化
- 提升内容专业度

### 4. 爆款复刻系统
- 分析热门内容
- 提供创作参考
- 辅助爆款制作

## 三、使用权限说明

### 3.1 账号管理

**账号类型：**
- **普通用户（user）**：公司授权员工，可使用所有创作功能
- **管理员（admin）**：系统管理员，可管理用户、查看数据统计

**账号创建流程：**
1. 员工向系统管理员申请开通权限
2. 管理员通过管理后台创建账号
3. 员工使用微信授权登录

### 3.2 权限控制机制

**技术保障：**
- ✅ 基于JWT的身份认证
- ✅ 用户白名单机制
- ✅ 数据隔离（用户只能查看自己的数据）
- ✅ 操作日志记录（所有操作可追溯）

**白名单管理：**
- 管理员维护授权员工白名单（基于微信openid）
- 只有白名单内的用户才能成功登录
- 员工离职后可立即取消权限

## 四、安全机制

### 4.1 数据安全
- 所有业务数据绑定用户ID，确保数据隔离
- 敏感操作需要管理员权限
- 完整的操作日志记录

### 4.2 访问控制
- 微信授权登录（确保身份真实性）
- 基于角色的权限控制（RBAC）
- 定期审计用户状态

### 4.3 审计功能
- 记录所有用户登录、查看、创建、修改、删除操作
- 管理员可查看用户操作日志
- 支持导出审计报告

## 五、管理后台功能

管理员可以通过管理后台进行以下操作：

### 5.1 用户管理
- 查看所有用户列表
- 启用/禁用用户账号
- 添加/移除管理员权限
- 导出用户信息

### 5.2 数据统计
- 用户活跃度统计
- 功能使用频率统计
- 内容创作数量统计

### 5.3 数据查看
- 查看用户的创作内容
- 查看用户的语料库
- 查看用户操作日志

### 5.4 共享管理
- 管理语料库共享权限
- 查看共享统计
- 批量设置全局共享

## 六、技术架构

### 6.1 前端技术
- Taro 4.x（支持微信小程序）
- React 18
- TypeScript
- Tailwind CSS

### 6.2 后端技术
- NestJS（企业级Node.js框架）
- Supabase（PostgreSQL数据库）
- JWT身份认证
- S3对象存储

### 6.3 第三方服务
- 微信小程序API（用户登录）
- 大语言模型（内容生成）

## 七、使用流程

### 7.1 员工首次使用流程
1. 向管理员申请开通权限
2. 管理员创建账号并添加到白名单
3. 员工微信搜索小程序
4. 微信授权登录
5. 开始使用

### 7.2 管理员操作流程
1. 使用管理员账号登录
2. 进入管理后台
3. 创建用户账号或管理现有用户
4. 配置语料库共享
5. 查看统计数据和操作日志
