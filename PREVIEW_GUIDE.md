# 小程序预览指南

## 🎯 为什么看不到UI界面？

你看到的是**小程序开发提示页面**，而不是小程序本身的UI界面。

**原因**：
- 小程序无法在网页浏览器中直接运行
- 小程序需要微信运行时环境
- 扣子平台的预览功能主要用于 H5 版本

---

## 📱 如何看到小程序UI界面

### 方式1：微信开发者工具预览（推荐）⭐

**步骤**：

1. **下载微信开发者工具**
   ```
   https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   ```

2. **导入项目**
   - 打开微信开发者工具
   - 选择"导入项目"
   - 项目路径：`/workspace/projects/dist-weapp`
   - AppID：在 `.env.local` 中查看

3. **查看界面**
   - 导入成功后，可以看到小程序的完整UI界面
   - 可以实时预览所有页面和功能
   - 可以调试代码、查看网络请求

4. **真机预览**
   - 点击工具栏的"预览"按钮
   - 生成二维码
   - 手机微信扫码查看真实效果

---

### 方式2：真机调试

1. 在微信开发者工具中点击"真机调试"
2. 手机扫描二维码
3. 可以在真机上查看和调试

---

### 方式3：体验版/正式版

1. 上传代码到微信后台
2. 设置为体验版
3. 添加体验成员
4. 手机微信扫码查看

---

## 💡 为什么不能在网页浏览器中预览？

**小程序 vs H5**：

| 特性 | 小程序 | H5 |
|------|--------|-----|
| 运行环境 | 微信客户端 | 浏览器 |
| 渲染引擎 | 微信引擎 | WebView |
| API 能力 | 微信原生API | 浏览器API |
| 预览方式 | 微信开发者工具 | 浏览器直接访问 |
| 扣子平台预览 | ❌ 不支持 | ✅ 支持 |

**结论**：
- 小程序必须在微信环境中运行
- 无法在扣子平台的网页预览中直接显示
- 需要使用微信开发者工具查看UI界面

## 📞 技术支持

如果遇到问题：

1. **检查 AppID 配置**：确保 `.env.local` 中有正确的 AppID
2. **检查构建产物**：确保 `dist-weapp/` 目录存在且完整
3. **检查微信开发者工具**：确保使用最新版本
4. **查看控制台错误**：在开发者工具中查看错误信息
