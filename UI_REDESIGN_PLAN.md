# 星厨房 UI 重构方案

## 一、整体布局结构

```
┌─────────────────────────────────────┐
│  顶部品牌区 (120px)                  │
│  Logo + 品牌名 + 用户按钮             │
├─────────────────────────────────────┤
│  核心功能区 (100px)                  │
│  灵感速记 (大卡片，渐变背景)           │
├─────────────────────────────────────┤
│  功能网格区 (2列布局)                │
│  ┌──────┐ ┌──────┐                  │
│  │功能1 │ │功能2 │                  │
│  └──────┘ └──────┘                  │
│  ┌──────┐ ┌──────┐                  │
│  │功能3 │ │功能4 │                  │
│  └──────┘ └──────┘                  │
│  ...                                │
├─────────────────────────────────────┤
│  底部导航栏 (固定)                   │
│  首页 | 工具箱 | 数据 | 我的          │
└─────────────────────────────────────┘
```

## 二、设计规范

### 1. 配色方案
```
主色：#3B82F6 (蓝色-500)
背景：#FFFFFF (白色)
次背景：#F8FAFC (slate-50)
文字主：#1E293B (slate-800)
文字次：#64748B (slate-500)
边框：#E2E8F0 (slate-200)

功能色系：
- 客资管理：#3B82F6 (蓝)
- 厨具回收：#10B981 (绿)
- 知识分享：#8B5CF6 (紫)
- 选题策划：#06B6D4 (青)
- 内容创作：#6366F1 (靛蓝)
- 语料优化：#14B8A6 (青绿)
- 爆款复刻：#EC4899 (粉)
- 直播数据：#F59E0B (琥珀)
```

### 2. 间距系统
```
页面边距：20px (px-5)
卡片间距：12px (gap-3)
卡片内边距：20px (p-5)
顶部区间距：16px
底部留白：80px (避开导航栏)
```

### 3. 图标规范
```
来源：lucide-react-taro
尺寸：24px (大) / 20px (中)
风格：线性图标
背景：48x48px 圆角方块，浅色背景
```

### 4. 字体规范
```
品牌标题：24px 粗体
功能区标题：18px 半粗
卡片标题：16px 半粗
卡片描述：12px 常规
导航文字：12px 常规
```

## 三、组件设计

### 1. 顶部品牌区
- 白色背景
- 左侧：Logo (24x24) + 品牌名 (24px 粗体) + 英文 (12px)
- 右侧：用户头像/登录按钮 (40x40 圆角方块)
- 高度：120px (含安全区)
- 无阴影，底部 1px 边框分隔

### 2. 核心功能卡片
- 渐变背景：from-blue-500 to-cyan-500
- 尺寸：全宽 x 100px
- 圆角：16px (rounded-2xl)
- 内部：图标 (56x56) + 标题 + 描述 + 箭头
- 阴影：shadow-lg
- 点击：scale-[0.98]

### 3. 功能网格卡片
- 白色背景
- 尺寸：48% 宽度 (2列布局)
- 圆角：12px (rounded-xl)
- 阴影：shadow-sm
- 内部结构：
  ```
  ┌──────────────┐
  │ 图标 (48x48) │
  │ 标题 (16px)  │
  │ 描述 (12px)  │
  └──────────────┘
  ```
- 点击：scale-[0.98]

### 4. 底部导航栏
- 白色背景
- 高度：56px + 安全区
- 顶部 1px 边框
- 4个Tab，均分宽度
- 图标：24px，文字：12px
- 未选中：灰色 #94A3B8
- 选中：蓝色 #3B82F6

## 四、视觉层次

### 重要性分级
1. **最高优先级**：核心功能卡片（灵感速记）
   - 渐变背景 + 大尺寸 + 阴影
   
2. **次优先级**：功能网格卡片
   - 白色背景 + 轻阴影 + 统一尺寸
   
3. **辅助层级**：顶部品牌区
   - 简洁白色背景

### 颜色权重
- 品牌色：渐变蓝青色（主功能）
- 功能色：每个功能独立颜色标识
- 中性色：背景、文字、边框

## 五、交互规范

### 点击反馈
- 所有可点击元素：active:scale-[0.98]
- 过渡时间：transition-transform duration-150
- 导航栏：切换动画

### 悬停状态
- 卡片：轻微阴影变化
- 按钮：透明度变化

### 加载状态
- 骨架屏：灰色占位
- 文字：渐变动画

## 六、响应式适配

### 手机端 (375px)
- 页面边距：20px
- 卡片宽度：48%
- 图标：48x48px
- 字体：标准大小

### 平板端 (768px)
- 页面边距：24px
- 卡片宽度：30% (3列)
- 图标：56x56px
- 字体：略大

## 七、实现要点

### 文件结构
```
src/pages/index/
├── index.tsx          # 页面主逻辑
├── index.config.ts    # 页面配置
└── components/
    ├── Header.tsx     # 顶部品牌区
    ├── QuickCard.tsx  # 核心功能卡片
    └── GridCard.tsx   # 功能网格卡片
```

### 样式原则
- 使用 Tailwind CSS 原子类
- 避免内联样式
- 颜色使用 CSS 变量
- 间距使用 Tailwind 类

### 性能优化
- 图标按需加载
- 长列表虚拟滚动
- 图片懒加载
- 避免过度动画

## 八、开发顺序

1. 创建页面基础结构
2. 实现顶部品牌区
3. 实现核心功能卡片
4. 实现功能网格卡片
5. 实现底部导航栏
6. 调整间距和细节
7. 测试交互效果
